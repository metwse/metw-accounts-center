§metw-accounts-center state

This crate contains the state, side effects, of the service crate. External integrations and data repository is served by this crate.

§Setup Recommendations

  • The token consumption and email rate limiting is enforced by Redis, in case of Redis state loss, one-time tokens can be accepted again. Make sure you have enabled persistent storage in Redis.
